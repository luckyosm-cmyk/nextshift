# NextShift Features

# 1. Calendar

The main screen of the application.

Functions:

- Monthly calendar
- Weekly calendar
- Daily timeline
- Infinite scrolling
- Swipe between months
- Today shortcut

---

# 2. Shift Management

Users can create:

- Morning shift
- Evening shift
- Night shift
- Custom shift

Each shift contains:

- Start time
- End time
- Break duration
- Notes
- Color

Recurring shifts:

- Every week
- Every 2 weeks
- Custom cycle

---

# 3. Vacation

Users can add:

- Vacation
- Sick leave
- Personal leave
- Public holiday
- Day off

---

# 4. Salary Calculator

- Hourly wage

- Daily salary
- Weekly salary

- Yearly salary
